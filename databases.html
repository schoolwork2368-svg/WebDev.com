Databases

A database is an organized collection of data, generally stored and accessed electronically from a computer system. In web development, databases are used on the backend to store and manage application data, such as user profiles, blog posts, product information, and more.

SQL (Relational) Databases

SQL (Structured Query Language) databases are relational databases. They organize data into tables with rows and columns, and they enforce a predefined schema. This structure makes them reliable and great for applications that require high data integrity.

Examples include: PostgreSQL, MySQL, Microsoft SQL Server, and SQLite.

NoSQL (Non-relational) Databases

NoSQL databases do not use the traditional table-based relational model. They come in various types, including document, key-value, wide-column, and graph. They are known for their flexibility, scalability, and ability to handle large volumes of unstructured data.

Examples include:

  • Document: MongoDB, CouchDB
  • Key-Value: Redis, DynamoDB
  • Graph: Neo4j
